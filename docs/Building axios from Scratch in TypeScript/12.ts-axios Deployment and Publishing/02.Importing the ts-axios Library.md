# Importing the ts-axios Library

## Importing in a TS Project

We use the [vue-cli](https://cli.vuejs.org/) scaffold to create a TypeScript Vue project, and then copy over a [demo](https://cn.vuejs.org/v2/guide/computed.html#%E4%BE%A6%E5%90%AC%E5%99%A8) snippet from the Vue official site that uses axios to send requests.

First we need to run `npm install ts-axios-new` to install the `ts-axios` library.

`HelloWorld.vue`

```vue
<template>
  <div class="hello">
    <p>
      Ask a yes/no question:
      <input v-model="question">
    </p>
    <p>{{ answer }}</p>
  </div>
</template>

<script lang="ts">
  import Vue from 'vue'
  import _ from 'lodash'
  import axios from 'ts-axios-new'

  export default Vue.extend({
    name: 'HelloWorld',
    data () {
      return {
        question: '',
        answer: 'I cannot give you an answer until you ask a question!'
      }
    },
    created () {
      this.debouncedGetAnswer = _.debounce(this.getAnswer, 500)
    },
    methods: {
      debouncedGetAnswer () {
        // do nothing
      },
      getAnswer () {
        if (this.question.indexOf('?') === -1) {
          this.answer = 'Questions usually contain a question mark. -)'
          return
        }
        this.answer = 'Thinking...'
        const instance = axios.create()
        instance.interceptors.request.use((config) => {
          config.params = {
            _t: +new Date()
          }
          return config
        })

        instance.get('https://yesno.wtf/api')
          .then((response) => {
            this.answer = _.capitalize(response.data.answer)
          })
          .catch((error) => {
            this.answer = 'Error! Could not reach the API. ' + error
          })
      }
    },
    watch: {
      question: function (newQuestion: string, oldQuestion: string) {
        this.answer = 'Waiting for you to stop typing...'
        this.debouncedGetAnswer()
      }
    }
  })
</script>

<!-- Add "scoped" attribute to limit CSS to this component only -->
<style scoped>
  h3 {
    margin: 40px 0 0;
  }

  ul {
    list-style-type: none;
    padding: 0;
  }

  li {
    display: inline-block;
    margin: 0 10px;
  }

  a {
    color: #42b983;
  }
</style>
```
This code mainly provides an `input` field bound to the `question` variable. When we type, it triggers a change in `question`, which runs the logic in `watch question` and calls the `this.debouncedGetAnswer` method. In effect, `debounce` invokes the `getAnswer` method to send the request.

We load the `ts-axios` library via `import axios from 'ts-axios-new'`. This actually imports `node_modules/ts-axios-new/dist/axios.es5.js`, because the `module` field configured in `ts-axios-new`'s `package.json` is `dist/axios.es5.js`. In `webpack`, `import` looks for the `module` field first, and falls back to the `main` field.

> Tip: When a library we import runs into problems at runtime, we can debug the corresponding imported code inside node_modules.

Note that here we first use the `axios.create()` method to create an `instance`, then add a request interceptor that, before every request is sent, adds a `_t` parameter whose value is a timestamp. We then call `instance.get` to send a request.

We can see that the whole demo runs correctly without any type-related issues, which shows that both the bundled code and the type declaration files of our library are problem-free.

## Importing in a JS Project

The TS library we wrote can still be imported by pure JS projects. This time we'll modify the code from the [《Rebuilding the Ele.me App with Vue.js 2.5 + cube-ui》](https://coding.imooc.com/class/74.html) course, changing the previous reference to `axios` into a reference to `ts-axios-new`. The course source code is open source, so those who haven't bought the course can also download it from [GitHub](https://github.com/ustbhuangyi/vue-sell).

First we need to run `npm install ts-axios-new` to install the `ts-axios` library, then modify the code.

`api/helpers.js`：

```javascript
import axios from 'ts-axios-new'

const urlMap = {
  development: '/',
  production: 'http://ustbhuangyi.com/sell/'
}
const baseUrl = urlMap[process.env.NODE_ENV]
const ERR_OK = 0

export function get(url) {
  return function(params = {}) {
    return axios.get(baseUrl + url, {
      params
    }).then((res) => {
      const {errno, data} = res.data
      if (errno === ERR_OK) {
        return data
      }
    }).catch((e) => {
    })
  }
}

```

We just need to change `import axios from 'axios'` to `import axios from 'ts-axios-new'`.

Next we run the project and find that it runs successfully. Since we implemented all of `axios`'s browser-side features, we can confidently make the swap.
