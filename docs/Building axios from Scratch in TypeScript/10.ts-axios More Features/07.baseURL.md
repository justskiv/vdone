# baseURL

## Requirements Analysis

Sometimes we make requests to multiple endpoints under a certain domain, and we don't want to write the full URL every time we send a request. Instead, we'd like to configure a `baseURL` so that afterwards we can just pass relative paths. For example:

```typescript
const instance = axios.create({
  baseURL: 'https://some-domain.com/api'
})

instance.get('/get')

instance.post('/post')
```

Once we configure `baseURL`, the `url` passed in for subsequent requests will be concatenated with our `baseURL` to form a complete absolute address, unless the `url` passed in for the request is already an absolute address.

## Implementation

First, let's modify the type definitions.

`types/index.ts`:

```typescript
export interface AxiosRequestConfig {
  // ...
  baseURL?: string
}
```

Next, let's implement two helper functions.

`helpers/url.ts`:

```typescript
export function isAbsoluteURL(url: string): boolean {
  return /^([a-z][a-z\d\+\-\.]*:)?\/\//i.test(url)
}

export function combineURL(baseURL: string, relativeURL?: string): string {
  return relativeURL ? baseURL.replace(/\/+$/, '') + '/' + relativeURL.replace(/^\/+/, '') : baseURL
}
```

Finally, let's call these two helper functions.

`core/dispatchRequest.ts`:

```typescript
function transformURL(config: AxiosRequestConfig): string {
  let { url, params, paramsSerializer, baseURL } = config
  if (baseURL && !isAbsoluteURL(url!)) {
    url = combineURL(baseURL, url)
  }
  return buildURL(url!, params, paramsSerializer)
}
```

## Writing the Demo

```typescript
const instance = axios.create({
  baseURL: 'https://img.mukewang.com/'
})

instance.get('5cc01a7b0001a33718720632.jpg')

instance.get('https://img.mukewang.com/szimg/5becd5ad0001b89306000338-360-202.jpg')
```

This demo is very simple: we request two images from imooc.com. Note that when the second request's `url` is already an absolute address, we no longer concatenate the `baseURL`.
