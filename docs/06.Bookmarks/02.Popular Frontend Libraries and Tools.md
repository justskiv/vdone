## Common Effect Components
- [Animate.css](https://animate.style/) Animation library
- [Swiper](https://www.swiper.com.cn/) Carousel component
- [mescroll](http://www.mescroll.com) Pull-to-refresh and infinite scroll framework — built on vanilla JS

## Utility Libraries
- [Lodash.js](https://www.lodashjs.com/)
- [Day.js](https://dayjs.fenxianglu.cn/) Date handling
- [Timeago.js](https://github.com/hustcc/timeago.js) Relative time, e.g. "N hours ago"
- [Echarts](https://echarts.apache.org/zh/index.html) Baidu charting library
- [Meditor.md](https://pandao.github.io/editor.md/) Open-source online Markdown editor
- [validator.js](https://github.com/validatorjs/validator.js) Validation library

 ## Vue Utility Libraries
- [vue-draggable](https://www.itxst.com/vue-draggable/tutorial.html) Vue drag-and-drop plugin based on Sortable.js
- [vue-qr](https://www.npmjs.com/package/vue-qr) Text-to-QR-code generator
- [vue-cropper](https://github.com/xyxiao001/vue-cropper) Image cropping plugin
- [vue-lazyload](https://www.npmjs.com/package/vue-lazyload) Lazy loading
- [vue-simple-upload](https://github.com/saivarunk/vue-simple-upload) Upload component
