# VDone

A VuePress theme for building structured knowledge bases, technical documentation and blogs.

Hard fork of [vuepress-theme-vdoing](https://github.com/xugaoyi/vuepress-theme-vdoing) by Evan Xu.

## Features

1. A developer-friendly `knowledge management tool`
2. Build a `structured` knowledge base — organize your content like a well-structured book
3. Blog functionality with `fragmented` knowledge format and personalized blog configuration
4. `Simple and efficient` — Markdown-centered project structure with built-in automation tools. Less configuration, more results. Multi-dimensional indexing for quick access to any topic

## Use Cases

* Knowledge base + blog
* Blog only
* Knowledge base only
* Documentation site

## Quick Start

```bash
# clone the project
git clone https://github.com/justskiv/vdone.git

# enter the project directory
cd vdone

# install dependencies
npm install # or yarn install

# develop
npm run dev # or yarn dev
```
