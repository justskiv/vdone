The 2-Minute Rule

Whenever you find it hard to start a task, try scaling it down to a 2-minute version.

  • Read a book -> Read one page
  • Write an article -> Write one sentence
  • Run 10 kilometers -> Put on your running shoes
  • Do 100 push-ups -> Do 1 push-up
  • Eat more fruits and vegetables -> Eat one piece of fruit
  • Write a program -> Write one function -> Write one line of code

The purpose is to make getting started super easy, so you just start. Once you begin (which may be the hardest step), momentum kicks in and you'll likely keep going.

  • Read one page -> Read 10 pages -> Finish the first chapter
  • Write one sentence -> Write the opening -> Write the body
  • Put on your running shoes -> Walk for 5 minutes -> Run for 5 minutes

Once you start, continuing becomes much easier. Sometimes you might even find that you've completed the entire task without realizing it.
